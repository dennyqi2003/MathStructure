# Natural Language

## Goal

$(R^{-1})^{-1} = R$

## Proof

- Let $(x, y) \in (R^{-1})^{-1}$. By definition of inverse, $(y, x) \in R^{-1}$, which implies $(x, y) \in R$. Hence, $(R^{-1})^{-1} \subseteq R$.

- Conversely, let $(x, y) \in R$. Then $(y, x) \in R^{-1}$, so $(x, y) \in (R^{-1})^{-1}$. Hence, $R \subseteq (R^{-1})^{-1}$.

Therefore, $(R^{-1})^{-1} = R$.
